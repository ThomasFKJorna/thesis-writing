* Surjective / Onto

A [[id:1c7c5c98-9acf-44e1-8921-4bb34eed0f92][function/mapping]] $f:A \rightarrow B$ is called *surjective* or *onto* if every element in $B$ is the [[id:b6b8ec07-b1e0-4b35-b7cc-1ce42947f146][Image]] of some element in $A$.

\[
\forall b\in B \quad \exists a \in A:\quad b=f(a)
\]

Note that this does not mean that each $a$ uniquely maps to an $f(a)$, for instance the function $f: \mathbb{R} \rightarrow \mathbb{R} = x^2$ maps both $1$ and $-1$ $\in A$ to $1$ $\in B$.

This happens if the function is both surjective and [[id:c615103d-1e0a-4b27-917c-70d935e267a6][Injective / one-to-one]], and it is then called [[id:1b978913-f923-48e8-b1e7-d0a42527dd5b][Bijective / One-to-one-correspondence]].
