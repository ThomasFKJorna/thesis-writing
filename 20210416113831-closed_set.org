* Closed set

** Metric space
Given a [[file:20210416111039-metric_space.org][Metric Space]] $(X,d)$, a subset $A\subseteq X$ is called *closed* if its [[file:20210416113423-complement.org][Complement]] $A^c$ is an [[file:20210416112309-open_set.org][Open set]].

** Topological space

Given a [[file:20210403182645-topological_space.org][Topological space]] $(X, T)$, a subset $A \subseteq X$ is called *closed* if the set $X-A$ is open.
