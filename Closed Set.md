# Closed set

## Metric space

Given a [[Metric Space]] $(X,d)$, a subset $A\subseteq X$ is called **closed** if its [[Complement]] $A^c$ is an [[Open set]].

## Topological space

Given a [[Topological space]] $(X, T)$, a subset $A \subseteq X$ is called **closed** if the set $X-A$ is open.
